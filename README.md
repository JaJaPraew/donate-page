# 💙 Support JaJa - Landing Page

หน้าเว็บ Landing Page มินิมอลสำหรับรวมช่องทางการสนับสนุน (Donation) และโซเชียลมีเดียต่างๆ เพื่อใช้ประกอบการสตรีมมิ่ง

🌍 **ดูหน้าเว็บของจริงได้ที่นี่:** [jajapraew.github.io](https://jajapraew.github.io)

## ✨ ฟีเจอร์หลัก (Features)
* ดีไซน์สะอาดตา ใช้งานง่ายสไตล์มินิมอล
* รองรับการแสดงผลทุกหน้าจอ (Responsive Design) ไม่ว่าจะดูผ่านมือถือหรือคอมพิวเตอร์
* มี QR Code สำหรับสแกนโดเนทได้ทันที
* ปุ่มลิงก์ไปยังหน้าสตรีมหลักและช่องทางอื่นๆ

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* HTML5
* CSS3
* โฮสต์เว็บไซต์ผ่าน GitHub Pages

## 🚀 วิธีการนำไปใช้งาน (How to Use)
ถ้าใครอยากได้หน้าเว็บแบบนี้ไปใช้ สามารถทำตามนี้ได้เลย:
1. กดปุ่ม `Fork` ที่มุมขวาบนของ Repository นี้
2. เปลี่ยนรูป `image.png` เป็น QR Code ของตัวเอง
3. เข้าไปแก้ลิงก์ในไฟล์ `index.html` ให้เป็นช่องทางของคุณ
4. เปิดใช้งาน GitHub Pages ในแถบ Settings
